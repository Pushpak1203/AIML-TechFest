Code Free Creation

🚀 Code Free Creation is a competition where participants build projects using no-code or low-code platforms.

📜 Rules & Regulations

  • Participants must use no-code or low-code platforms.
  • Each team can have up to 3 members.
  • Time limit: 2 hours.
  • Projects will be judged on creativity and functionality.

The competition lasts for 2 hours.
